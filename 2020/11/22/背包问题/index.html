01背包问题

题目描述

题目描述来自背包问题九讲
有$N$件物品和一个容量为$V$的背包。放入第i件物品耗费的费用是$C_i$，得到的价值是$W_i$。求解将哪些物品装入背包可使价值总和最大。

状态转移方程

我们定义一个数组$dp[i][j]$，每一个元素表示当容量为$j$的时候从前$i$个物品中选择所能得到的最大商品价值，下面列出状态转移方程。

如何理解这个状态转移方程？
当$j＜C[i]$时，即当前的容量不足以放进第i号物品，那么第$i$个不能选了，当然就只能选前$i-1$个，这个背包的最大价值就是只考虑前$i-1$个物品能构造出的最大价值。
而当$j\ge C[i]$时，当前的容量可以放进第i号物品。那么我们就要考虑是要往背包里塞第$i$号物品呢还是不塞第$i$号物品。
要塞入第$i$号物品的话，价值就是$d[i-1][j-C[i]]+W[i]$，因为书包要塞下这个物品就得先腾出它的空间，在腾出它的空间后的书包的最大价值加上第$i$号物品就是选择塞入第$i$号物品的书包价值。
如果选择不塞入第$i$号物品，这时候的情况就相当于在前$i-1$个物品里选择了，那么这时书包的价值就是$d[i-1][j]$
最后在塞入第$i$号物品和不塞入第$i$号物品种选择价值更大的即可。

一道例题

P1048 采药

题目描述

辰辰是个天资聪颖的孩子，他的梦想是成为世界上最伟大的医师。为此，他想拜附近最有威望的医师为师。医师为了判断他的资质，给他出了一个难题。医师把他带到一个到处都是草药的山洞里对他说：“孩子，这个山洞里有一些不同的草药，采每一株都需要一些时间，每一株也有它自身的价值。我会给你一段时间，在这段时间里，你可以采到一些草药。如果你是一个聪明的孩子，你应该可以让采到的草药的总价值最大。”

如果你是辰辰，你能完成这个任务吗？

输入格式

第一行有 22 个整数 $TT(1 \le T \le 10001≤T≤1000)$和 $MM(1 \le M \le 1001≤M≤100)$，用一个空格隔开，TT 代表总共能够用来采药的时间，MM 代表山洞里的草药的数目。

接下来的 MM 行每行包括两个在 11 到 100100 之间（包括 11 和 100100）的整数，分别表示采摘某株草药的时间和这株草药的价值。

输出格式

输出在规定的时间内可以采到的草药的最大总价值

一道题目

P1616 疯狂的采药

题目描述

LiYuxiang 是个天资聪颖的孩子，他的梦想是成为世界上最伟大的医师。为此，他想拜附近最有威望的医师为师。医师为了判断他的资质，给他出了一个难题。医师把他带到一个到处都是草药的山洞里对他说：“孩子，这个山洞里有一些不同种类的草药，采每一种都需要一些时间，每一种也有它自身的价值。我会给你一段时间，在这段时间里，你可以采到一些草药。如果你是一个聪明的孩子，你应该可以让采到的草药的总价值最大。”

如果你是 LiYuxiang，你能完成这个任务吗？

此题和原题的不同点：

  1. 每种草药可以无限制地疯狂采摘。
  2. 药的种类眼花缭乱，采药时间好长好长啊！师傅等得菊花都谢了！

    输入格式

    输入第一行有两个整数，分别代表总共能够用来采药的时间$t$和代表山洞里的草药的数目$m$。
    第$2$到第 $(m + 1)$行，每行两个整数，第 $(i + 1)$行的整数$a_i$,$b_i$分别表示采摘第 $i$ 种草药的时间和该草药的价值。

    输出格式

    输出一行，这一行只包含一个整数，表示在规定的时间内，可以采到的草药的最大总价值

    北航OJ：斯卡里欠刮痧之王

    题目描述

    斯卡里欠骰子王亨利为了得到一个头盔，已经在竞技场决斗了三天了。

众所周知，在决斗中有很多的战术动作可以选择，一些战术动作可以最大限度地保护自己，但只能对敌人造成少量伤害（俗称“刮痧”），另一些战术动作则会完全放弃保护自己，在自己受到重创的同时对敌人造成大量伤害（俗称“自爆”）。

现在，王亨利的血量为n，他一共可以做k个战术动作，对于第i个战术动作，使用这个战术动作可以对敌人造成xi点伤害，同时自己会受到yi点伤害（即血量会减少yi）。

对于每个战术动作，王亨利可以使用任意次数，那么，在他倒下之前，最多能对敌人造成多少伤害。

王亨利倒下当且仅当他使用了一个战术动作t，yt大于等于他的当前血量n0（此战术动作对敌人造成的伤害仍然要计算在对敌人造成的总伤害中）

输入

第一行包含一个整数T，为数据的组数。

对于每组数据：

第一行包含两个正整数n,k，含义同题目描述。

接下来k行，每行包含两个正整数x,y，表示一个战术动作的参数，含义同题目描述。

输出

对于每组数据，输出一行：

如果王亨利能对敌人造成无限的伤害，输出字符串Gua Sha Zhi Wang!

其它情况输出一个整数，表示王亨利最多能对敌人造成多少伤害
