本地springboot后端配置

给队友们写的指南

Posted by Charlot on 2021-11-30

本地后端需要准备：

  • 一个能跑java的IDE（建议就用IDEA）

  • mysql数据库

  • 一个可视化数据库的插件（可以再IDEA里装database插件，也可以下载例如navicat这样的软件）

  • （可选）postman，用以测试后端函数

IDEA

JetBrains(IEDA、CLion、Pycharm等)针对学生提供了正版免费使用资格。但需要完成学生认证，其中邮箱需要填含有edu的邮箱（比如北航邮箱，亲测有效）。学生认证可以按照这篇博客，操作比较简单：

https://blog.csdn.net/qq_45656879/article/details/104606348

完成认证后即可免费使用以下所有软件：

(不过学生认证有效期是一年，第二年快到期时会有邮箱提示，重新验证即可

解压提供的academic_springboot的zip，进入文件夹，选中里面的academic_system，右键以idea project的形式打开。

信任本项目

接下来等待indexing完成

如果出现了set up SDK的提示（此时所有的java文件的图标不是蓝色而是橙色的），则去设置一下项目SDK，设置为1.8就可以

设置完成后橙色的图标会变成蓝色。

接下来如果出现很多报错，则点开右侧的maven栏，点击刷新即可。

接下来请看一下AcademicSystemApplication是否可以运行，如果可以的话点击小三角即可

或者进到AcademicSystemApplication点击main函数旁边的小三角

出现以下提示说明后端启动成功（不用在意顶端的爆红）

如果你更改过数据库root的密码，请在这里进行更改

mysql

这一部分是照搬的数据库里提供的教材，如果你已经安装好了mysql请跳过。

Windows 10 | MySQL 8.0

https://dev.mysql.com/doc/refman/8.0/en/windows-installation.html

https://www.microsoft.com/en-us/download/details.aspx?id=48145

https://dev.mysql.com/downloads/installer/

  1. 安装MySQL 8.0 需要先安装好 Microsoft Visual C++ 2015 Redistributable 。⾸先从 开始 菜单 -> 设置 -> 应⽤ -> 应⽤和功能 找到已安装应⽤列表，搜索是否已安装 Visual C++ 2015 。

如果没有安装，则需要从 这个⻚⾯ 下载，两个⽂件都要选择，点击“Next”开始下载，然后分别点击安装。

  1. 从 这个⻚⾯ 下载 Windows 安装包，下载第 2 个安装包 (422.4M) ，然后点击安装。

  1. 选择默认的“Developer Default”，点击“Next>”。

点击“Execute”，安装结束后，点击“Next>“。

点击 ”Next>“ ，从这⾥开始始终保持默认即可。

设置 root ⽤户密码，下⽅可以添加其他⽤户，点击“Next>”。

可以设置是否开机启动 MySQL，我这⾥保持默认，点击“Next>”

点击“Execute”，结束后点击 “Finish”。

接下来的⻚⾯都保持默认。

这⾥输⼊之前设定的 root 密码，导⼊示例数据库。

接下来的⻚⾯保持默认。

安装结束。

加载提供的sql数据

postman（可选）

下载地址

postman是用来测试后端函数的一个东西

对于post函数，选好方法，输入函数路径，选择body里的row，按照对象的形式写好参数，点击send就可以查看后端返回的样式。

对于get函数用的是form-data
